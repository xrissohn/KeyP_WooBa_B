# Changelog

All notable changes to KeyP will be documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and the project intends to use [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Root MIT license and open-source community documentation
- Contribution, security, and code-of-conduct policies
- Structured bug-report, feature-request, and pull-request templates
- Bilingual project overview and release-readiness documentation

## [0.1.0] - 2026-07-10

### Added

- Natural-language interest planning with deterministic fallback
- NAVER, X, RSS/Atom, grounded AI search, SerpAPI, and YouTube connectors
- Relevance and source-credibility review for new candidates
- Shared source cache, provider budgets, deduplication, and baseline suppression
- Cursor-based subscription feeds, bookmarks, webhook ingestion, and FCM delivery
- Firebase Installation ID and App Check integration
- Kotlin Multiplatform mobile client foundations
- OpenAPI 3.1 specification and automated contract checks
- CI with type checking, builds, tests, and coverage thresholds
